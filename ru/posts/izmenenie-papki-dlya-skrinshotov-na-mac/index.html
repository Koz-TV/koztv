Изменение папки для скриншотов на Mac

2018-02-01

По умолчанию все скриншоты сохраняются на рабочий стол, но вы можете изменить это расположение на любую другую папку на Mac.

Для начала запустите терминал.

Лично я предпочитаю iTerm, но вы можете использовать стандартный терминал, который установлен на каждом Mac.

Выберите папку, куда хотите сохранять скриншоты, и введите следующую команду в окне терминала:

defaults write com.apple.screencapture location

Теперь перетащите папку для скриншотов в окно терминала. Тогда команда будет выглядеть так:

defaults write com.apple.screencapture location ~/Documents/Screenshots

Теперь нужно перезапустить SystemUIServer, введите эту команду в терминале:

killall SystemUIServer

Вот и всё 😉

Сделайте скриншот прямо сейчас с помощью Cmd + Shift + 3, и новый скриншот сохранится в выбранную папку.
